The OntoWiki site extension

  • Heino, N.; Tramp, S. & Auer, S.: Managing Web Content using Linked Data Principles -- Combining semantic structure with dynamic content syndication, IEEE Computer Society, 2011
  • Extends OntoWiki to serve HTML pages for resources (if client wants HTML)
  • Homepage: https://github.com/AKSW/site.ontowiki

Main Resource Content

  • Not all content can be structured as plain literals and relations
  • We also need a main content where we can use markup
    • sioc:content, dc:description, … do not allow markup
    • We introduce site:content here.
  • Please use dc:abstract and other non-markup properties as well, to allow better syndication.
    • They are required by some templates anyway (e.g. project and group)

Helper Markup

  • Very often we need dynamic content inline in the main content
  • The site extension introduces helper markup (wiki actions) which can be used inline in any literal:
    • Link to other resources / integrate images
    • Generate list by using inline queries
    • Output content of other resources
    • … some more markup helper here
  • Syntax: {{name p1="v1" p2="v2" ...}}

Markup Helper: query

Markup Helper: query

Markup Helper: link

  • {{link uri="http://aksw.org/About"}}
    • give a full uri, link title is fetched
  • {{link uri="foaf:Person"}}
    • give a qname, link title is fetched
  • {{link text="our projects" literal="Projects"}}
    • give a literal for search and a text for the link
  • Other Parameters: property, class, prefix, suffix, iprefix, isuffix, direct

Other markup helper

  • toc: generate table of contents
  • img: integrate images
  • literal: integrate literals from any resource
  • …

Documentation available here: https://github.com/AKSW/site.ontowiki/wiki

Current State: Classes

  • foaf:Document: a basic web page without specific lists and queries
  • foaf:Group: a sub group with lists of member, research areas and projects
  • foaf:Person: a person with lists of projects and research areas
  • foaf:Project: an AKSW project, which can be a funded, a community or an open-source project

Current State: General Properties

  • site:menuLabel If the resource title is too long for the menu, use this property in addition to [[rdfs:label]].
  • dcterms:abstract This is a plain text attribute (no markup).
  • aksw:hookline This a less than 100 character description of the resource.
  • sioc:feed Use this object property to relate a RSS or Atom feed to this resource.
